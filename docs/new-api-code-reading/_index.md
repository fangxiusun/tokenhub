## 核心架构要点

### 1. 分层架构
```
HTTP Request → Middleware → Router → Controller → Service → Model → Relay → Upstream Provider
```

### 2. 适配器模式 (relay/channel/)
- 统一的 `Adaptor` 接口支持 40+ AI 提供商
- 工厂模式动态实例化适配器
- 每个提供商独立包，易于扩展

### 3. 计费系统
- 两阶段结算：预扣费 → 结算 → 退款
- 支持钱包和订阅两种资金源
- 表达式计费支持动态定价

### 4. 渠道路由
- Ability 表实现 (group, model, channel_id) 映射
- 支持优先级、权重、亲和性
- 自动健康检查和故障转移

### 5. 前端架构
- React 19 + TanStack Router 文件路由
- Zustand 状态管理
- 嵌入式 SPA 部署
