# hash.go 代码阅读文档

## 1. 全局总结
该文件是密码学哈希函数的工具模块，提供了 SHA-256 和 SHA-1 哈希计算功能。主要包含五个函数：`Sha256Raw`、`Sha1Raw`、`Sha1`、`HmacSha256Raw` 和 `HmacSha256`。这些函数用于数据完整性验证、数字签名、密码哈希等安全相关场景。

## 2. 依赖关系
- **标准库依赖**：
  - `crypto/hmac`: HMAC（基于哈希的消息认证码）实现
  - `crypto/sha1`: SHA-1 哈希算法
  - `crypto/sha256`: SHA-256 哈希算法
  - `encoding/hex`: 十六进制编码/解码
- **内部依赖**：
  - 无（本文件是自包含的工具模块）

## 3. 类型定义
本文件没有定义新的类型。所有函数都使用基本类型：
- `[]byte`: 字节切片
- `string`: 字符串类型

## 4. 函数详解

### 4.1 Sha256Raw(data []byte) []byte
- **功能**：计算数据的 SHA-256 原始哈希值
- **参数**：`data []byte` - 要计算哈希的数据
- **返回值**：`[]byte` - 32 字节的 SHA-256 哈希值
- **实现逻辑**：
  1. 创建 SHA-256 哈希计算器
  2. 写入数据
  3. 返回计算结果（不包含长度前缀）

### 4.2 Sha1Raw(data []byte) []byte
- **功能**：计算数据的 SHA-1 原始哈希值
- **参数**：`data []byte` - 要计算哈希的数据
- **返回值**：`[]byte` - 20 字节的 SHA-1 哈希值
- **实现逻辑**：
  1. 创建 SHA-1 哈希计算器
  2. 写入数据
  3. 返回计算结果

### 4.3 Sha1(data []byte) string
- **功能**：计算数据的 SHA-1 哈希值并返回十六进制字符串
- **参数**：`data []byte` - 要计算哈希的数据
- **返回值**：`string` - 40 字符的十六进制字符串
- **实现逻辑**：
  1. 调用 `Sha1Raw` 获取原始哈希值
  2. 使用 `hex.EncodeToString` 转换为十六进制字符串

### 4.4 HmacSha256Raw(message, key []byte) []byte
- **功能**：计算 HMAC-SHA256 原始值
- **参数**：
  - `message []byte`: 消息数据
  - `key []byte`: 密钥
- **返回值**：`[]byte` - 32 字节的 HMAC-SHA256 值
- **实现逻辑**：
  1. 使用密钥创建 HMAC-SHA256 计算器
  2. 写入消息数据
  3. 返回计算结果

### 4.5 HmacSha256(message, key string) string
- **功能**：计算 HMAC-SHA256 并返回十六进制字符串
- **参数**：
  - `message string`: 消息字符串
  - `key string`: 密钥字符串
- **返回值**：`string` - 64 字符的十六进制字符串
- **实现逻辑**：
  1. 将字符串转换为字节切片
  2. 调用 `HmacSha256Raw` 计算原始 HMAC
  3. 使用 `hex.EncodeToString` 转换为十六进制字符串

## 5. 关键逻辑分析

### 5.1 哈希算法选择
- **SHA-256**: 256 位哈希，提供较高的安全性，适用于大多数场景
- **SHA-1**: 160 位哈希，安全性较低但仍广泛使用，可能用于兼容性
- **HMAC-SHA256**: 基于 SHA-256 的消息认证码，提供数据完整性和身份验证

### 5.2 原始值与十六进制字符串
- **Raw 函数**: 返回原始字节切片，适用于二进制数据处理
- **非 Raw 函数**: 返回十六进制编码的字符串，适用于文本表示和日志记录
- 这种设计提供了灵活性，可以根据使用场景选择合适的格式

### 5.3 密码学安全性
- 使用标准库的密码学实现，确保算法正确性
- HMAC 使用密钥和消息的组合，防止长度扩展攻击
- 原始哈希函数不包含密钥，适用于纯数据完整性验证

### 5.4 性能考虑
- 每次调用都创建新的哈希计算器，适合一次性计算
- 对于大量数据，可能需要考虑流式处理或复用计算器
- 十六进制编码会增加内存使用（2倍大小）
