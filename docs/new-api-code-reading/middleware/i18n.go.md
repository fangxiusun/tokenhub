# i18n.go 代码阅读文档

## 1. 全局总结
i18n.go 实现了国际化中间件，从请求头提取语言设置并注入到上下文中。

## 2. 依赖关系
### 2.1 导入的包
- `constant` - 常量定义
- `i18n` - 国际化包
- `gin-gonic/gin` - Web 框架

### 2.2 被引用的文件
- `router/main.go` - 路由初始化

## 3. 类型定义
### 3.1 函数
- `I18n() gin.HandlerFunc` - 国际化中间件

## 4. 函数详解
### 4.1 I18n
- **签名**: `func I18n() gin.HandlerFunc`
- **职责**: 解析 Accept-Language 头并设置上下文语言
- **逻辑流程**:
  1. 从 Accept-Language 头提取语言
  2. 标准化语言代码
  3. 设置到上下文 ContextKeyLanguage

## 5. 关键逻辑分析
- **语言优先级**: 用户设置 > Accept-Language > 默认语言
- **标准化**: 处理语言代码变体（如 zh-CN, zh-TW）

## 6. 关联文件
- `i18n/i18n.go` - 国际化核心
- `constant/context_key.go` - 上下文键定义
