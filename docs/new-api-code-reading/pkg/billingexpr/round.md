# round.go 代码阅读文档

## 1. 全局总结
该文件提供配额四舍五入函数 QuotaRound，用于将浮点数配额值转换为整数。采用“四舍五入远离零”的规则，确保所有计费路径（预消费、结算、分解验证、日志字段）使用一致的四舍五入逻辑，避免 ±1 的差异。

## 2. 依赖关系
- **math**: 使用 math.Round 函数进行四舍五入。

## 3. 类型定义
无类型定义。

## 4. 函数详解
### QuotaRound
将 float64 配额值转换为 int，使用 half-away-from-zero 四舍五入。例如：0.5 → 1，-0.5 → -1，0.4 → 0，-0.6 → -1。

## 5. 关键逻辑分析
- **一致性保证**: 所有涉及配额转换的代码必须使用此函数，确保四舍五入行为一致。
- **简单性**: 直接封装 math.Round，但提供了明确的语义和文档。

## 6. 关联文件
- **settle.go**: 在 ComputeTieredQuota 中调用 QuotaRound 进行配额四舍五入。
- **billingexpr_test.go**: 测试 QuotaRound 的各种边界情况。
