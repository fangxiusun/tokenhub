# constants.go 代码阅读文档

## 1. 全局总结

该文件定义了阿里通义万相视频生成频道的常量，包括支持的模型列表和频道名称。这些常量被同目录下的 `adaptor.go` 引用，用于模型验证和频道标识。

## 2. 依赖关系

无外部依赖，仅定义包级变量。

## 3. 类型定义

### 变量

| 变量名 | 类型 | 说明 |
|--------|------|------|
| `ModelList` | `[]string` | 阿里通义万相支持的视频生成模型列表 |
| `ChannelName` | `string` | 频道标识名称，值为 `"ali"` |

### 模型列表详情

| 模型名 | 说明 |
|--------|------|
| `wan2.5-i2v-preview` | 万相 2.5 preview（有声视频），推荐模型 |
| `wan2.2-i2v-flash` | 万相 2.2 极速版（无声视频） |
| `wan2.2-i2v-plus` | 万相 2.2 专业版（无声视频） |
| `wanx2.1-i2v-plus` | 万相 2.1 专业版（无声视频） |
| `wanx2.1-i2v-turbo` | 万相 2.1 极速版（无声视频） |

## 4. 函数详解

无函数定义。

## 5. 关键逻辑分析

该文件为纯常量定义文件，无逻辑代码。模型列表中的所有模型均为图生视频（i2v）类型，版本从 2.1 到 2.5 不等。注意 `wan2.5-i2v-preview` 是唯一标注为有声视频的模型。

## 6. 关联文件

- `relay/channel/task/ali/adaptor.go` — 引用 `ModelList` 和 `ChannelName`
